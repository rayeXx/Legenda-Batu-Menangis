# Legenda Batu Menangis 🪨

Animasi 2D interaktif **Legenda Batu Menangis** dibuat menggunakan [Processing](https://processing.org/).

**by Gacor Hebat Animation Studios**

---

## 📖 Tentang Proyek

Animasi ini menceritakan legenda rakyat Kalimantan tentang seorang anak perempuan yang durhaka kepada ibunya dan dikutuk menjadi batu.

## 🎬 Alur Scene

| Scene | Durasi | Keterangan |
|---|---|---|
| Scene 1 | 0:00 – 0:15 | Pembuka — Judul & Langit Fajar |
| Scene 2 | 0:15 – 0:44 | Kehidupan Desa |
| Scene 3 | 0:44 – 1:14 | Perjalanan ke Pasar |
| Scene 4 | 1:14 – 1:45 | Dialog di Pasar |
| Scene 5 | 1:45 – 2:15 | Doa Sang Ibu |
| Scene 6 | 2:15 – 2:45 | Kutukan |
| Scene 7 | 2:45 – 3:00+ | Penutup & Credits |

## 🚀 Cara Menjalankan

1. Install [Processing 3](https://processing.org/download)
2. Buka file `Legenda Batu Menangis.pde`
3. Klik tombol **Run (▶)**

## 👥 Tim

Lihat credits di dalam animasi (scene akhir).
